# What's Venus ?
Venus alerts you completion of the process.

# Setup
```
pip install pyaudio # or conda install
pip install synthesizer
```

# How to use
```
import venus

# ~~~ your code here ~~~
# after all above

venus.beep_bit(num=-1)
```
Venus plays music according to the number specified.
If set to `-1`, play randomly.
